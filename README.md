# Dos Attack Tool

A simple tool for testing DoS attacks (for educational purposes only!)

## Features
- Dos attack with multi session and change size of request

## Run

Download -> https://github.com/Why4ck/Dos/releases        
run installer.exe               
run cmd -> write why4ck         

use dos tool!

example for use (more details in cli tool): why4ck dos --url "http://127.0.0.1" --session 10 --concurrency 1 --size 15
10 session, 1 request size - 10 kb
